APPLAUSE BUTTON PLUGIN | Best "Facebook Like Button" alternative. Add this applause button to your website and let users like your content by "clapping". Shortcode: [wp-applause-button]

=== Description ===

**Add "Applause Button" to your website and users can like your content by "clapping". This plugin is also featuring rich snippet ratings which will show up as 5 star ratings in the search results. Install now & start getting 5 star claps!**


== Key Features ==
* Very lightweight.
* Responsive and customizable shortcodes.
* Insert shortcode directly from the post editor.
* Rich Snippet 5 star rating.


== Shortcodes ==

[wp-applause-button] default applause button

[wp-applause-button style="width:100px;height:100px;"] changed size

[wp-applause-button color="red"] changed color

[wp-applause-button multiclap="true"] multiple claps (by default it true)

Custom shortcode example:
[wp-applause-button color="red" style="width:100px;height:100px;" multiclap="true"]
